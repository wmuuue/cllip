# 剪贴板笔记 (ClipNotes)

一个功能强大的Android笔记应用，支持自动剪贴板监听、浮动窗口交互、音频录制和局域网设备同步。

## 功能特性

### 核心功能
- **浮动窗口** - 圆形浮动图标，可拖动并记忆位置
- **剪贴板监听** - 后台自动捕获复制的内容并保存到笔记
- **笔记弹窗** - 长按浮动图标显示笔记弹窗，支持透明度调节
- **笔记管理** - 查看、编辑、删除和清空笔记
- **音频录制** - 支持录音、播放和保存到笔记中
- **颜色自定义** - 分别设置剪贴板文字和用户输入文字的颜色
- **局域网发现** - 自动发现同一网络下的其他设备
- **笔记同步** - 将笔记发送到其他设备
- **数据持久化** - 使用SQLite数据库存储所有数据

### 智能功能
- **无障碍服务** - 支持通过无障碍服务在后台监控剪贴板
- **智能剪贴板控制** - 尊重用户设置，可随时暂停/恢复监听
- **浮动图标位置记忆** - 移动图标后自动保存位置，下次启动时恢复
- **快速保存** - 点击浮动图标直接保存剪贴板内容，app仅短暂显示后返回后台
- **一键复制** - 点击笔记条目即可复制到剪贴板
- **点击反馈** - 笔记弹窗中点击笔记后文字变为淡黄色
- **设备过滤** - 发送笔记时智能过滤自己的设备
- **接收确认** - 接收笔记前需要用户确认

## 技术栈

- **语言**: Kotlin 1.9.20

## 安装和使用

### 安装要求
- Android 7.0 (API 24) 或更高版本
- 需要以下权限：
  - 悬浮窗权限
  - 无障碍服务权限（可选，用于后台剪贴板监控）
  - 录音权限
  - 网络访问权限

### 首次使用
1. 安装APK并打开应用
2. 授予悬浮窗权限（显示浮动图标）
3. 可选：开启无障碍服务以获得更好的后台剪贴板监控
4. 应用会自动进入后台，浮动图标显示在屏幕上
5. 复制任何文本，会自动保存到笔记
6. 点击浮动图标保存当前剪贴板内容
7. 长按浮动图标查看笔记弹窗

### 功能使用

#### 浮动图标操作
- **点击** - 保存当前剪贴板内容到笔记
- **长按** - 显示笔记弹窗
- **拖动** - 移动图标位置（位置会自动保存）

#### 添加笔记
- 点击右下角的"+"按钮手动添加笔记
- 复制文本会自动添加到笔记
- 点击浮动图标保存剪贴板内容

#### 录制音频
- 点击左下角的麦克风按钮开始录音
- 再次点击停止录音并保存

#### 发送笔记到其他设备
1. 确保两台设备在同一WiFi网络
2. 两台设备都安装并运行此应用
3. 点击菜单 -> 发送
4. 选择目标设备（不会显示自己的设备）
5. 对方设备确认接收

#### 设置选项
1. 点击菜单 -> 设置
2. **剪贴板监控开关** - 控制是否自动保存剪贴板内容
3. **弹窗透明度** - 调节笔记弹窗的透明度（10%-100%）
4. **文字颜色** - 分别设置剪贴板文字和用户输入文字的颜色

## 权限说明

| 权限 | 用途 |
|------|------|
| SYSTEM_ALERT_WINDOW | 显示浮动窗口图标和笔记弹窗 |
| FOREGROUND_SERVICE | 后台运行剪贴板监听服务 |
| BIND_ACCESSIBILITY_SERVICE | 无障碍服务，用于后台剪贴板监控 |
| RECORD_AUDIO | 录制音频笔记 |
| INTERNET | 局域网设备发现和笔记同步 |
| ACCESS_WIFI_STATE | 检测网络状态 |

## 注意事项

1. **电池优化**: 建议在系统设置中关闭此应用的电池优化，以确保后台服务持续运行
2. **权限**: 首次使用需要授予悬浮窗权限和录音权限
3. **无障碍服务**: 开启无障碍服务可以更可靠地监控剪贴板，但需要在系统设置中手动开启
4. **剪贴板监控设置**: 关闭"启用剪贴板监控"后，无论是普通服务还是无障碍服务都不会保存剪贴板内容
5. **网络**: 设备同步功能需要两台设备在同一局域网
6. **数据**: 所有数据存储在本地SQLite数据库，卸载应用会丢失数据

## 常见问题

**Q: 浮动窗口不显示？**
A: 检查是否授予了悬浮窗权限，设置 -> 应用 -> 特殊权限 -> 显示在其他应用上层

**Q: 剪贴板监听不工作？**
A: 1. 确保"启用剪贴板监控"开关已开启
   2. 确保应用没有被系统强制关闭，关闭电池优化
   3. 可以开启无障碍服务以获得更可靠的监控

**Q: 找不到其他设备？**
A: 确保两台设备在同一WiFi网络，并且都安装并运行了此应用

**Q: 音频录制失败？**
A: 检查是否授予了录音权限

**Q: 浮动图标位置没有保存？**
A: 确保拖动图标后松手，位置会自动保存
